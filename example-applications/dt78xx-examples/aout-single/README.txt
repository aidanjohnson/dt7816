This example writes single value(s) to the analog output channel(s)

Usage : 
aout-single volt1 [volt2]

volt1 is the voltage output to analog output 0
volt2 applies to the DT7816 only. If specified, it is the voltage output to 
analog output 1
Volt value must be less than 10.0V and greater than or equal to -10.0V
